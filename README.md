# AI-Powered Student Career Roadmap Generator

A Streamlit web app that uses Google's Gemini LLM to generate a personalized career roadmap for students and beginners who want to become AI Engineers.

## How to run

1. Create virtual environment:

python -m venv .venv
.venv\Scripts\activate

text

2. Install packages:

pip install -r requirements.txt

text

3. Create a `.env` file with:

GOOGLE_API_KEY=your_gemini_api_key_here

text

4. Run the app:

streamlit run app.py
